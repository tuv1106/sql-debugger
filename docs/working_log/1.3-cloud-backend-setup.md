# Task 1.3: Cloud Backend Project Setup

## Plan

**Goal:** Set up the FastAPI cloud backend that handles AI reasoning.

**Files to create:**
- `cloud_backend/main.py` — FastAPI app with CORS and `/health` endpoint
- `cloud_backend/config.py` — environment variable configuration (ANTHROPIC_API_KEY, LLM_MODEL, session limits)
- `cloud_backend/__init__.py`, `api/__init__.py`, `agents/__init__.py`, `agents/tools/__init__.py`, `sessions/__init__.py`, `models/__init__.py` — package structure
- `cloud_backend/requirements.txt` — dependencies
- `cloud_backend/pyproject.toml` — pytest config
- `tests/cloud_backend/conftest.py` — test client fixture
- `tests/cloud_backend/test_health.py` — health endpoint tests
- `Dockerfile.cloud` — containerized deployment

**Tests (2):**
1. Health endpoint returns `{"status": "ok", "version": "0.1.0"}`
2. CORS allows configured origin

**Acceptance criteria:**
- `pytest` runs with 0 errors
- `/health` returns OK

## Retrospect

Implementation followed the plan exactly. CORS origins are read from the `ALLOWED_ORIGINS` env var (comma-separated), defaulting to localhost:5173 and localhost:3000.
