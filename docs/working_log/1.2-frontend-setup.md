# Task 1.2: Frontend Project Setup

## Plan

**Goal:** Set up the React/TypeScript frontend with 4-panel layout shell.

**Files to create:**
- Vite + React 19 + TypeScript scaffold
- `frontend/src/App.tsx` — 4-panel layout (Schema Browser, Query Console, AI Chat, Bottom Drawer)
- `frontend/src/components/SchemaPanel.tsx`, `QueryConsole.tsx`, `AiChatPanel.tsx`
- `frontend/src/stores/connection-store.ts` — Zustand store skeleton
- `frontend/src/api/client.ts` — API client with local/cloud base URLs
- `frontend/vitest.config.ts` — test configuration
- `frontend/src/App.test.tsx` — layout test
- `Dockerfile.frontend` — multi-stage build (node → nginx)

**Tests (1):**
1. App renders all 4 panels

**Acceptance criteria:**
- `npm run dev` starts dev server
- `npm run test` runs with 0 errors
- Docker image builds

## Retrospect

Deviated from plan in one area: initially put vitest config inside `vite.config.ts` using the `test` property, which caused `tsc -b` to fail during Docker build (TypeScript doesn't recognize the `test` field). Fixed by separating into a dedicated `vitest.config.ts`. Also discovered Node.js was installed via Homebrew but not linked to PATH — required `eval "$(/opt/homebrew/bin/brew shellenv)"` before npm commands.
